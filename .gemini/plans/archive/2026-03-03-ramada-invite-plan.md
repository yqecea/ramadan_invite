# Micro Landing Page — Қоныстой + Ауызашар

## 🎯 ДЛЯ ТЕБЯ (Простым языком)

### Что будет сделано?
- Сайт-приглашение, **заточенный под телефон** (десктоп тоже будет, но телефон — главный)
- Акварельные казахские орнаменты + иллюстрация Астаны
- Обратный отсчёт, переключатель языка, ссылки на карты
- Плавные анимации, оптимизированные для мобильных

### Что ты получишь?
HTML-файл, который открывается как приложение на телефоне — быстро, красиво, без тормозов.

### ⚠️ Может ли что-то сломаться?
Нет. Новый проект с нуля.

---

## Дизайн-система

| Параметр | Решение |
|----------|---------|
| **Приоритет** | 📱 Mobile-first (375px → 768px → 1440px) |
| **Шрифты** | Great Vibes (script, имя семьи) + Cormorant Infant (serif, заголовки) + system sans (body) |
| **Палитра** | Бежевый `#F5F0E8`, оливковый `#5C6B4F`, золотой `#C4A35A`, текст `#2C2C2C` |
| **Анимации** | GPU-only (`transform`, `opacity`), `prefers-reduced-motion`, spring-подобные easing |
| **Изображения** | WebP, lazy loading, srcset для разных экранов |

## Этапы работы

### Этап 1 — Фундамент
> Создать файлы, подключить шрифты, написать HTML-структуру и базовый CSS

**Файлы:**
- [NEW] `index.html` — семантичный HTML, мета-теги, Open Graph
- [NEW] `style.css` — CSS-переменные, mobile-first сетка, все секции
- [NEW] `script.js` — таймер + языковой переключатель + scroll-анимации

---

### Этап 2 — Визуал
> Сгенерировать изображения и вставить в страницу

- Акварельная панорама Астаны (hero)
- Казахские орнаменты (декоративные разделители)

---

### Этап 3 — Анимации (Mobile-optimized)
> Добавить scroll-анимации, hover-эффекты (touch-safe), микро-взаимодействия

- `IntersectionObserver` для появления секций
- Все анимации через `transform` + `opacity` (GPU)
- Таймер: flip-style карточки с мягким переходом

---

### Этап 4 — Полировка и проверка
> Протестировать на мобиле, проверить скорость, адаптив

- Открыть в Chrome DevTools (375px iPhone)
- Проверить KZ ↔ RU переключение
- Проверить ссылки Maps / 2GIS
